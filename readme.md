## Overview


This is a sub-150g antweight battle bot built for small-scale robot combat events. The goal was to make a robot that is lightweight, reliable, easy to repair, and competitive while staying within the 150g weight limit.

The robot uses a 3D-printed chassis, removable top armour, and modular electronics so damaged parts can be replaced quickly between fights.

## Design Goals

- Stay safely below the 150g event limit
- Keep the centre of gravity low
- Make damaged parts quick to remove and replace
- Protect the battery, receiver, and wiring
- Use readily available parts where possible
- Make it easy to improve after each fight
